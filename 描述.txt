页面设计:

主页:index.html
功能：登录注册,程序入口
提供数据：
	登录数据:用户名，密码
		格式：{
			'userId'	:	'userId',
			'passWord'  :	'passWord'
		}
	注册数据:用户名，昵称，密码，性别，年龄，描述，头像
		格式：{
			'userId'	:	'userId',
			'userName'	:	'userName',
			'passWord'	:	'passWord',
			'sex'		:	'sex',
			'age'		:	'age',
			'describe'	:	'describe',
			'display'	:	'display'
		}
需求数据:无

大厅：hotel.html
功能：
	已登录：
		显示已关注blog用户和推荐
	未登录：
		显示推荐
	搜索博主和博文
提供数据:
	无
需求数据：用户名(可选),推荐用户名,关注用户名,搜索用户名,搜索blogId
	格式:{
			'userId'	:	'userId',
			'userName'	:	'userName ',
			'fllowId'	:	'fllowId'
	}

personal:personal.html
功能:显示所选用户的博文目录 
提供数据:所选用户id
格式：{
	'userId'	: 	'userId'
}
需求数据:所选用户的所有blogId,所选用户display,博文数量
格式:{
		'userId'	:	'userId',
		'display'	:	'display',
		'blogNum'	:	'blogNum'
		'blog'		: 	[
							{
								'blogId'
							},
						]
}
